# Project Philosophy

This project was designed to demonstrate an end-to-end actuarial analytics workflow rather than isolated scripts or notebooks.

The documentation reflects the same philosophy:

- **Root README** — Project overview
- **Documentation** — Technical implementation
- **Power BI Guide** — Business presentation
- **Results Catalog** — Analytical outputs

This separation keeps the repository approachable for recruiters while providing sufficient technical depth for reviewers interested in the implementation.
